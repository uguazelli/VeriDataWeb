POLÍTICA DE PRIVACIDAD Y TÉRMINOS DE USO DE DATOS

Última actualización: 03 de Enero de 2026

1. Controlador de Datos

Esta aplicación y servicios asociados son operados por Veridata (y su división educativa VeriAcademy).
Sitio web oficial: www.veridatapro.com
Contacto: admin@veridatapro.com

2. Datos Recopilados

Para el funcionamiento de la integración con la API de WhatsApp Business y servicios de IA, recopilamos y procesamos los siguientes datos:

  • Número de teléfono (ID de WhatsApp).
  • Nombre de perfil (según lo disponible públicamente en WhatsApp).
  • Contenido de los mensajes enviados por el usuario para interactuar con nuestros servicios automatizados.
  • Registros técnicos de interacción (marca de tiempo, estado de entrega).

3. Finalidad del Procesamiento

Los datos se utilizan estrictamente para:

  • Entregar respuestas automatizadas y contenido solicitado (Cursos, Soporte, Servicios).
  • Procesamiento de lenguaje natural (IA) para una mejor comprensión de las solicitudes.

No vendemos, alquilamos ni compartimos sus datos con terceros para fines de marketing no solicitado.

4. Compartir con Terceros

Para la ejecución técnica del servicio, los datos transitan por las infraestructuras de:

  • Meta Platforms, Inc. (Proveedor de la API de WhatsApp).
  • OpenAI / Anthropic / Google (Proveedores de LLM, si corresponde, para procesamiento de IA).
  • Proveedores de Cloud Hosting (AWS/Azure/GCP) donde reside nuestra infraestructura.

5. Retención y Eliminación de Datos

Los datos se mantienen mientras sean necesarios para la prestación del servicio. El usuario puede solicitar la eliminación completa de sus datos en cualquier momento enviando el comando "ELIMINAR DATOS" en el chat o enviando un correo electrónico al contacto anterior.

6. Seguridad

Utilizamos cifrado en tránsito (HTTPS/TLS) para todas las comunicaciones entre la API de WhatsApp y nuestros servidores.
