2. Collected Data

To operate the WhatsApp Business API integration and AI services, we collect and process the following data:

  • Phone number (WhatsApp ID).

3. Purpose of Processing

Data is used strictly to:

  • Deliver automated responses and requested content (Courses, Support, Services).

We do not sell, rent, or share your data with third parties for unsolicited marketing purposes.

4. Sharing with Third Parties

For technical service execution, data transits through the infrastructures of:

  • Meta Platforms, Inc. (WhatsApp API Provider).
  • OpenAI / Anthropic / Google (LLM Providers, if applicable, for AI processing).
  • Cloud Hosting Providers (AWS/Azure/GCP) where our infrastructure resides.

5. Data Retention and Erasure

Data is retained as long as necessary for service provision. The user can request complete deletion of their data at any time by sending the command "DELETE DATA" in the chat or by sending an email to the contact above.

6. Security

We use encryption in transit (HTTPS/TLS) for all communications between the WhatsApp API and our servers.
