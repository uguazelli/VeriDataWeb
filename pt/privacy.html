POLÍTICA DE PRIVACIDADE E TERMOS DE USO DE DADOS

Última atualização: 03 de Janeiro de 2026

1. Controlador de Dados

Este aplicativo e serviços associados são operados pela Veridata (e sua divisão educacional VeriAcademy).
Website oficial: www.veridatapro.com
Contato: admin@veridatapro.com

2. Dados Coletados

Para o funcionamento da integração com a API do WhatsApp Business e serviços de IA, coletamos e processamos os seguintes dados:

  • Número de telefone (ID do WhatsApp).
  • Nome de perfil (conforme disponibilizado publicamente no WhatsApp).
  • Conteúdo das mensagens enviadas pelo usuário para interagir com nossos serviços automatizados.
  • Logs técnicos de interação (timestamp, status de entrega).

3. Finalidade do Processamento

Os dados são utilizados estritamente para:

  • Entregar as respostas automatizadas e conteudos solicitados (Cursos, Suporte, Serviços).
  • Processamento de linguagem natural (IA) para melhor entendimento das solicitações.

Nós não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins de marketing não solicitado.

4. Compartilhamento com Terceiros

Para a execução técnica do serviço, os dados transitam pelas infraestruturas de:

  • Meta Platforms, Inc. (Provedor da API do WhatsApp).
  • OpenAI / Anthropic / Google (Provedores de LLM, se aplicável, para processamento de IA).
  • Provedores de Cloud Hosting (AWS/Azure/GCP) onde nossa infraestrutura reside.

5. Retenção e Exclusão de Dados

Os dados são mantidos enquanto forem necessários para a prestação do serviço. O usuário pode solicitar a exclusão completa de seus dados a qualquer momento enviando o comando "EXCLUIR DADOS" no chat ou enviando um e-mail para o contato acima.

6. Segurança

Utilizamos criptografia em trânsito (HTTPS/TLS) para todas as comunicações entre a API do WhatsApp e nossos servidores.
